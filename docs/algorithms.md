# アルゴリズム概論

## アルゴリズムとは
問題を解決するための手順や計算方法のことです。
入力に対して、有限の手順を経て、求められる出力を得るための手続きです。

## アルゴリズムの評価指標

### 時間計算量 (Time Complexity)
アルゴリズムの実行にかかる時間が、入力サイズ $n$ に対してどのように増加するかを表す指標です。
一般的に、ビッグオー記法 ($O$) を用いて表します。

- $O(1)$: 定数時間。入力サイズに関わらず一定。
- $O(\log n)$: 対数時間。二分探索など。
- $O(n)$: 線形時間。線形探索など。
- $O(n \log n)$: 準線形時間。効率的なソート（クイックソート、マージソート）など。
- $O(n^2)$: 二乗時間。単純なソート（バブルソート）など。
- $O(2^n)$: 指数時間。

### 空間計算量 (Space Complexity)
アルゴリズムの実行に必要なメモリ領域が、入力サイズ $n$ に対してどのように増加するかを表す指標です。
こちらもビッグオー記法を用いて表します。
