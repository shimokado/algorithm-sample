# バブルソート (Bubble Sort)

隣り合う要素の大小を比較し、順序が逆であれば交換する操作を繰り返すことで、整列を行う単純なソートアルゴリズムです。

## アルゴリズムの概要
1. 配列の末尾から隣接する要素を比較する。
2. 大小関係が逆（昇順なら 右 < 左）であれば交換する。
3. これを配列の先頭まで繰り返すと、最小の要素が先頭に移動する（泡が浮かび上がるように見える）。
4. 確定した先頭要素を除いて、同様の操作を繰り返す。

## 計算量
- **時間計算量**: $O(n^2)$
  - 最良ケース: すでにソートされている場合 $O(n)$ （改良版の場合）
  - 最悪ケース: 逆順にソートされている場合 $O(n^2)$
- **空間計算量**: $O(1)$ （内部ソート）

## 擬似コード
```
function bubble_sort(list):
    n = length(list)
    for i from 0 to n - 1:
        for j from 0 to n - 1 - i:
            if list[j] > list[j + 1]:
                swap(list[j], list[j + 1])
    return list
```
