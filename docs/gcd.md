# 最大公約数 (GCD: Greatest Common Divisor)

2つの整数 $a, b$ の最大公約数を求めるアルゴリズムです。

## ユークリッドの互除法
2つの自然数 $a, b$ ($a \ge b$) について、$a$ を $b$ で割った余りを $r$ とすると、$a$ と $b$ の最大公約数は $b$ と $r$ の最大公約数に等しいという性質を利用します。

### 手順
1. $a$ を $b$ で割った余り $r$ を求める。
2. $r = 0$ ならば、$b$ が最大公約数である。
3. $r \ne 0$ ならば、$a \leftarrow b, b \leftarrow r$ として 1 に戻る。

### 計算量
- 時間計算量: $O(\log (\min(a, b)))$
