# 挿入ソート (Insertion Sort)

整列済みの部分列に対して、新たな要素を適切な位置に挿入していくことでソートを行うアルゴリズムです。

## アルゴリズムの概要
1. 配列の2番目の要素から開始する（1番目の要素はソート済みとみなす）。
2. 現在の要素を取り出し、ソート済み部分（左側）の要素と比較する。
3. ソート済み部分の要素が現在の要素より大きい場合、その要素を右にずらす。
4. 適切な位置が見つかるか、ソート済み部分の先頭に達したら、そこに現在の要素を挿入する。
5. 配列の末尾まで繰り返す。

## 計算量
- **時間計算量**:
  - 最悪: $O(n^2)$ （逆順にソートされている場合）
  - 最良: $O(n)$ （すでにソートされている場合）
  - 平均: $O(n^2)$
- **空間計算量**: $O(1)$ （内部ソート）
- **安定性**: 安定なソート

## 擬似コード
```
function insertion_sort(list):
    n = length(list)
    for i from 1 to n - 1:
        key = list[i]
        j = i - 1
        while j >= 0 and list[j] > key:
            list[j + 1] = list[j]
            j = j - 1
        list[j + 1] = key
    return list
```
