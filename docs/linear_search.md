# 線形探索 (Linear Search)

リストや配列の先頭から順に要素を比較して、目的の要素を探し出す最も単純な探索アルゴリズムです。

## アルゴリズムの概要
1. 配列の先頭から順に、探索対象の値と比較する。
2. 値が一致すれば、そのインデックス（または存在すること）を返す。
3. 配列の最後まで探索しても見つからなければ、見つからなかったこと（-1 や null など）を返す。

## 計算量
- **時間計算量**: $O(n)$
  - 最良ケース: 先頭で見つかる場合 $O(1)$
  - 最悪ケース: 最後まで見つからない、または末尾にある場合 $O(n)$
- **空間計算量**: $O(1)$

## 擬似コード
```
function linear_search(list, target):
    for index from 0 to length(list) - 1:
        if list[index] == target:
            return index
    return -1
```
