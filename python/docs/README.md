## 🎯 学習の進め方

### ステップ1: Python基礎を理解する
まず基礎編（00〜02）を読んで、Pythonの基本構文とJavaScriptとの違いを理解しましょう。

### ステップ2: 簡単なモジュールから始める
- **線形探索**や**バブルソート**など、シンプルなアルゴリズムから始めることをお勧めします
- コードを読みながら、Pythonの構文に慣れていきましょう

### ステップ3: より複雑なモジュールへ
- **グラフアルゴリズム**や**マージソート**など、より高度な実装に進みます
- 再帰やクラスなど、Python特有の書き方を学びます

### ステップ4: 実際に動かしてみる
- 各モジュールのテストファイル（`tests/`フォルダ）を見て、実際の使用例を確認しましょう
- 自分でコードを書いて実験してみましょう

## ❓ よくある質問

### Q: JavaScriptとPythonの最大の違いは何ですか？
A: インデントがコードの構造を決める点、型ヒントの存在、そしてPython特有のショートハンド（リスト内包表記、タプルアンパックなど）です。

### Q: どのドキュメントから読み始めればいいですか？
A: `00_python_basics.md`から始めて、`01_python_data_types.md`、`02_control_flow.md`の順に進むことをお勧めします。

### Q: このコードを実行するには？
A: プロジェクトルートで`pytest`を実行するか、個別のモジュールを`import`して使用できます。詳細は[README.md](../README.md)を参照してください。
